# Byrock Hoof X-ray Portal

Interactive equine laminitis and venogram visual aid for trial image review.

## Contents

- `index.html` - the deployable single-page diagnostic visualisation.
- `assets/reference-images/` - staged reference radiograph images.
- `assets/trial-images/` - trial radiograph image set used by the portal.

## Local Preview

Open `index.html` directly in a browser, or serve the folder locally:

```bash
python3 -m http.server 4179
```

Then visit `http://127.0.0.1:4179/`.
